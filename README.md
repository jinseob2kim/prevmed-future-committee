# 예방의학회 미래세대위원회 3분과


## 1회: 2026.03.30 16:00
- 참석자: 강민구, 김진섭(분과장), 박준영, 서강, 양정호, 오연우
- 목적: 분과 운영 방향 정리, 분과장 선출, 수련교육과정 개편 인식조사 연구 기본 틀 논의, IRB 준비 방향 및 일정 협의
- 핵심 문제의식: 실제 진로·직무와 수련 목표의 불일치, 차별화된 고유 경험 부족, 파트별/순환형 수련 부족, 현장·실무 경험 부족, 연구 중심 고착 가능성
- 주요 결정: 김진섭 선생님을 분과장으로 선출하고, 인식조사는 우선 전공의 및 전문의 중심의 IRB 기반 연구로 추진하기로 함
- 설문 방향: 기존 설문을 수련환경 중심으로 재구성하고, 전공의 수첩 및 공식 역량 문서를 참고해 중요 역량과 실제 교육 간 불일치를 드러내는 문항을 포함하기로 함
- 후속 과제: 박준영 선생님이 약 40문항 내외의 설문 1차 초안을 작성하고, 양정호 선생님은 IRB 준비를 지원하며, 강민구 위원장은 예산 및 응답 보상 가능 범위를 확인하기로 함
- 일정: 공식 다음 회의는 약 1개월 후 개최를 목표로 하고, 그 전에 서울권 참석자 중심의 비공식 논의를 진행하기로 함
